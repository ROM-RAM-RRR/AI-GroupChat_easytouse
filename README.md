这是一个非常简单的AI聊天群组！运行起来也很简单！

下载(右上角Code-Download ZIP)后，先需要在.env(可以用VSCode打开)中粘贴你的API_KEY与BASE_URL，并以管理员身份运行configure.bat以进行环境配置（需要已有python），然后运行run.bat文件即可进入聊天界面。

默认群组配置是两只deepseek（以api形式调用），当然也可以手动加入其他成员

English:
This is a very simple AI chat group! It's also very easy to run!

After downloading, first paste your API_KEY and BASE_URL in the .env file (which can be opened with VSCode).
Run configure.bat as an administrator to set up the environment (Python must already be installed).
Then run the run.bat file to enter the chat interface.

The default group configuration consists of two DeepSeek models (called via API), but you can also manually add other members.

Modifying the AI behavior rules in the left sidebar can give them character traits and relationships.
